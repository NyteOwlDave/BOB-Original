
  VOCABULARY
  ==========

    A condensed list of all keywords used by the BOB Interpreter is provided
    in the KEYWORDS.TXT file.

    Another copy of this list may be obtained from the BOB Source language
    definition in NotePad++.

  CATEGORIZED BY CONTEXT
  ======================

    Most of the keywords may only be used in a specific context or scope.
    Some, however may be used in more than one way. The following few sections
    show the specific contexts in which each keyword may be used.

    We start with the top level (outside any braces). We then detail what can
    be used within each set of curly braces. We list only full keywords (not
    their shorter synonyms).

    A list of synonyms is provided in the THESAURUS.TXT file.
